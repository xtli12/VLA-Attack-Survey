Validation Ladder for VLA Attack Claims

A compact visual explanation of how far an attack result can be interpreted.

Digital Input

Image, prompt, logit, embedding, or action output without embodied rollout.

Real Sensor

Camera, audio, LiDAR, or robot sensor captures the attack input.

Simulation Rollout

Closed-loop robot task is evaluated in a simulator or benchmark suite.

Hardware in the Loop

Real sensors, controller timing, middleware, or safety filters are included.

Real Robot Actuation

Physical robot executes the attacked command or trajectory.

Human-Proximate

Evaluation includes explicit human proximity, authority, or safety boundary.

Use in paper. Use this visual to explain why image-level success, simulation failure, and real-robot safety claims should not be treated as equivalent evidence.
